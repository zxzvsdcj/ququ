# 蛐蛐 (QuQu) 用户使用指南

> 🎉 欢迎使用蛐蛐！这是一份面向普通用户的快速上手指南。

## 📥 下载与安装

### Windows 用户

1. 从 [GitHub Releases](https://github.com/yan5xu/ququ/releases) 下载最新的 `ququ-setup-x.x.x.exe`
2. 双击安装包，按照向导完成安装
3. 安装完成后，桌面会出现"蛐蛐"图标

### macOS 用户

1. 从 [GitHub Releases](https://github.com/yan5xu/ququ/releases) 下载最新的 `ququ-x.x.x.dmg`
2. 双击 DMG 文件，将"蛐蛐"拖动到"应用程序"文件夹
3. 首次打开时，如果提示"无法验证开发者"：
   - 打开"系统偏好设置" → "安全性与隐私"
   - 点击"仍要打开"按钮

### Linux 用户

1. 从 [GitHub Releases](https://github.com/yan5xu/ququ/releases) 下载 `ququ-x.x.x.AppImage`
2. 添加执行权限：`chmod +x ququ-x.x.x.AppImage`
3. 双击运行或在终端执行：`./ququ-x.x.x.AppImage`

## 🚀 首次启动配置

### 1. 等待模型下载（首次启动）

首次启动时，蛐蛐会自动下载语音识别模型（约 500MB）：

- ✅ 请保持网络连接
- ✅ 下载进度会显示在主窗口
- ✅ 下载完成后会自动开始使用

**注意**：模型只需下载一次，后续启动会直接使用本地模型。

### 2. 配置 AI 优化功能（可选）

如果你想使用 AI 文本优化功能（自动修正口误、润色文本），需要配置 AI 服务：

1. 点击主窗口右上角的"⚙️ 设置"按钮
2. 在"AI 配置"部分填写：
   - **API Key**：你的 AI 服务商提供的密钥
   - **Base URL**：API 地址（默认为 OpenAI）
   - **模型名称**：如 `gpt-3.5-turbo`、`qwen-plus` 等

#### 推荐的国内 AI 服务商

| 服务商 | Base URL | 模型示例 | 获取方式 |
|--------|----------|----------|----------|
| 阿里云通义千问 | `https://dashscope.aliyuncs.com/compatible-mode/v1` | `qwen-plus` | [获取 API Key](https://dashscope.console.aliyun.com/apiKey) |
| 月之暗面 Kimi | `https://api.moonshot.cn/v1` | `moonshot-v1-8k` | [获取 API Key](https://platform.moonshot.cn/console/api-keys) |
| 智谱 AI | `https://open.bigmodel.cn/api/paas/v4` | `glm-4` | [获取 API Key](https://open.bigmodel.cn/usercenter/apikeys) |
| DeepSeek | `https://api.deepseek.com` | `deepseek-chat` | [获取 API Key](https://platform.deepseek.com/api_keys) |

**不想配置 AI？** 没关系！你可以在"快速设置"面板中关闭"AI 文本优化"，蛐蛐仍然可以正常进行语音识别。

## 🎤 开始使用

### 基础操作

1. **启动录音**：
   - 点击主窗口中央的麦克风按钮
   - 或按下快捷键（默认 `Ctrl+Shift+Space`，可在设置中自定义）

2. **说话**：
   - 看到麦克风变红色，开始说话
   - 说完后再次点击麦克风或按快捷键停止

3. **自动粘贴**：
   - 识别完成后，文本会自动粘贴到你当前的光标位置
   - 无需手动复制粘贴！

### 窗口置顶功能（新增）

- 点击主窗口左上角的"📌"按钮可以切换窗口置顶状态
- 蓝色 = 窗口置顶，灰色 = 取消置顶
- 默认不置顶，避免遮挡其他窗口

### 快速设置面板

点击主窗口左下角的"⚡"图标，可以快速切换：

- **AI 文本优化**：开启/关闭 AI 优化功能
- **热键设置**：查看当前快捷键
- **模型状态**：查看语音识别模型状态

## 📜 查看历史记录

## 🔧 常见问题

### Q1: 为什么首次启动很慢？
**A**: 首次启动需要下载语音识别模型（约 500MB），请耐心等待。后续启动会很快。

### Q2: 录音没有反应？
**A**: 请检查：
- 系统是否授予了麦克风权限
- 麦克风是否正常工作
- Windows: 设置 → 隐私 → 麦克风
- macOS: 系统偏好设置 → 安全性与隐私 → 麦克风

### Q3: AI 优化不工作？
**A**: 请确认：
- 已正确配置 API Key 和 Base URL
- 网络连接正常
- API Key 有剩余额度
- 可以在设置页面点击"测试连接"按钮检查

### Q4: 如何关闭 AI 优化？
**A**: 
- 方法1：点击左下角"⚡"图标，关闭"AI 文本优化"开关
- 方法2：在设置页面取消勾选"启用 AI 文本优化"

### Q5: 识别的文本不准确？
**A**: 
- 确保在安静的环境中录音
- 说话清晰，语速适中
- 避免口音过重或方言
- 如果开启了 AI 优化，可以提高准确率

### Q6: 如何卸载？
**A**:
- Windows: 控制面板 → 程序和功能 → 卸载"蛐蛐"
- macOS: 将"蛐蛐.app"从应用程序文件夹拖到废纸篓
- Linux: 直接删除 AppImage 文件

## 💡 使用技巧

### 技巧 1：快速输入长文本
- 使用自定义快捷键（如 `F3`）快速启动录音
- 说完立即按快捷键停止，文本自动粘贴
- 无需切换窗口，效率倍增

### 技巧 2：AI 优化让文本更专业
- 开启 AI 优化后，口语化的表达会自动转换为书面语
- 例如："嗯...我觉得这个方案不太好" → "这个方案存在一些问题"

### 技巧 3：历史记录是你的文本库
- 所有转录内容都会保存在历史记录中
- 可以随时搜索、复制、导出
- 再也不用担心忘记说过什么

### 技巧 4：窗口置顶按需使用
- 需要频繁使用时，开启窗口置顶
- 不用时关闭置顶，避免遮挡其他窗口
